# MDPyCrack
A 1-5 character MD5 cracker built with Python 3

## Features
* Brute force attack (planned)
* Dictionary attack

## Limitations
Restricted to:
* Unsalted
* MD5
* 1-5 characters long
* ASCII characters only

## Usage
The following command prints the usage information
```python
python3 cracker.py -h
```
